# TabiMap
現在地にメモを残すアプリ。
タイトル、内容、評価を記録することができる。

# 説明
1. 現在地の位置情報を元に地図上にメモを残すことができる　。
2. 現在地の位置情報からGooglePlacesApiを元に自動で最寄り施設の写真を表示することが可能。
3. 自分のメモを編集削除可能。
